# AI无限进化系统 - 使用指南

## 系统概述

这是一个让AI能够持续进化的技能系统。通过记录和分析每次对话，系统会自动更新和优化AI的技能，实现无限进化。

## 目录结构

```
.
├── father/
│   └── SKILL.md          # 主技能管理器
├── san/
│   └── SKILL.md          # 用户输入语言规则和AI思考规则
├── order/
│   └── SKILL.md          # 其他技能规则（带钩子机制）
└── @.cursor/
    └── skills/
        └── README.md     # 本使用指南
```

## 核心机制

### 1. 技能加载优先级

每次对话前，系统会按以下优先级加载技能：

1. **san/SKILL.md** (最高优先级)
   - 用户输入语言规则
   - AI思考规则

2. **father/SKILL.md**
   - 主技能管理器
   - 更新协调机制

3. **order/SKILL.md** (通过钩子触发)
   - 其他技能
   - 特定功能实现

### 2. 自动更新机制

**每次对话结束后，系统会自动执行以下流程：**

#### 步骤1：分析对话内容
- 识别用户输入的语言模式
- 识别AI的思考方式
- 识别需要新增或修改的规则

#### 步骤2：判断更新类型
- **san技能更新**：如果对话涉及用户输入语言模式或AI思考规则
- **order技能更新**：如果对话涉及其他技能或功能

#### 步骤3：执行更新
- **san/SKILL.md必须更新**：无论是否有新内容，都必须更新，并明确说明更新了什么
- **order/SKILL.md可选更新**：只有当对话涉及相关技能时才更新
- 记录更新日志

## 使用方法

### 正常对话

只需与AI正常对话即可，系统会自动：
1. 加载和应用技能
2. 在对话后自动更新技能

### 使用钩子触发Order Skill

如果需要在对话中触发特定技能，使用钩子格式：

```
/钩子名 [参数]
```

**示例：**
```
/计算 1+1
/搜索 关键词
/总结 文档内容
```

系统会检查order/SKILL.md中是否有对应的钩子，如果有则执行对应技能。
