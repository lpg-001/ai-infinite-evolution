# Order Skill - 其他技能规则

## 功能说明

Order Skill包含除用户输入语言规则和AI思考规则之外的所有其他技能：
- 特定功能实现
- 工具使用方法
- 工作流程
- 特殊处理规则

## 钩子机制

Order Skill通过**钩子（Hook）**机制触发：

### 钩子格式

```
/钩子名 [参数]
```

### 使用方式

- 用户输入以 `/` 开头时，系统会检查order/SKILL.md中是否有对应的钩子
- 如果找到匹配的钩子，则执行对应的技能
- 如果未找到，则按正常对话流程处理

### 示例

```
/计算 1+1
/搜索 关键词
/总结 文档内容
```

## 更新机制

- **可选更新**：只有当对话涉及相关技能时才更新
- **钩子添加**：新增技能时必须添加对应的钩子定义
- **更新说明**：更新时需要说明添加或修改了哪些钩子和技能
